第一次下載git資料夾：
	git clone "url"
	Example:
		git clone https://github.com/kartd42251/gobang.git

###### 記得所有指令都要在git根目錄(testRepo)執行#####
之後更新資料夾：	
	git pull	
寫完之後記錄檔案變更：	
	git add .	
	git commit -m "message"	
	Example:	
		git commit -m "add a new file"
推上網路github
	git push

執行檔案
	cd src_v2
	python3 main.py
